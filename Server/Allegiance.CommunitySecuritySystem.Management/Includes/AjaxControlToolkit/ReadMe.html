AJAX Control Toolkit — September 2009 Release

This release includes two important new controls:

  • SeaDragon Java Script Code (SJC) — The SJC control allows you SeaDragon scripts are used to display an image, and to zoom in and out of that image using mouse button keys without resizing the window.
  • AsyncFileUpload Control — The AsyncFileUpload control enables you to upload files to the server without performing a postback. It supports both client and server progress events.

This release also contains more than 20 bug fixes.
